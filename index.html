Front Double Biceps

▼
자세
  • 심판 향해 정면으로 서기

이두근 크기·봉우리이두근 전·후 분할전완근삼각근흉근복부허벅지종아리

Back Double Biceps

Back Lat Spread

Side Triceps
사이드 트라이셉스
▼
자세
  • 우·좌 선택하여 옆으로 서기

  • 가까운 발 → 발바닥 평평하게 (평발)
  • 먼 발 → 발가락으로 지탱
  • 앞팔 압력으로 삼두근 수축, 가슴 올리고 복부·허벅지·종아리 수축

체스트: 가까운 발 = 발가락 | 트라이셉스: 가까운 발 = 평발

#포즈명손 위치핵심 포인트
1프론트 포지션한 손 복직근(허리)한 발 옆으로 살짝 / 머리·눈 정면 / 똑바로 서기
2쿼터 턴 라이트 (왼편)오른손 복직근왼팔 중심선 뒤 아래 / 왼 엉덩이 살짝 올리기 / 왼 무릎 구부려 발가락 지탱
3쿼터 턴 백한 손 복직근한 발 옆 / 상체 똑바로 / 자연스러운 척추전만
4쿼터 턴 라이트 (오른편)왼손 복직근오른팔 중심선 뒤 아래 / 오른 엉덩이 살짝 올리기 / 오른 무릎 구부려 발가락 지탱

2·4
Quarter Turn Right 상세
왼편 / 오른편 (방향만 반대)
▼
왼편 (2번)
  • 오른손은 복직근에 / 왼팔은 신체 중심선보다 뒤로 아래로 내리기
  • 왼쪽 엉덩이를 약간 올리고
  • 왼쪽 다리 무릎 약간 구부리고 발가락으로 지탱
오른편 (4번)
  • 왼손은 복직근에 / 오른팔은 중심선보다 뒤로 아래로 내리기
  • 오른쪽 엉덩이 약간 올리고, 오른 무릎 구부려 발가락 지탱
⚠ 광배근 최대 수축 (보디빌더식) 금지 — 자연스러운 펼침만 허용
3
Quarter Turn Back 상세
쿼터 턴 백
▼
  • 한 손 복직근에 / 한 발 옆으로 살짝
  • 상체를 앞으로 기울이지 않고 똑바로 세우기
  • 허리 아랫부분 자연스럽게 굽히거나 약간 척추전만 형태
⚠ 상체를 심판 쪽으로 돌릴 수 없음 — 무대 뒤쪽 바라봐야 함
5
I-Walking
아이 워킹
▼

동작 우아함속도·몸짓무대 존재감카리스마자연스러운 리듬개인 매력

열사병 (Heat Stroke)
체온 40°C↑의식 혼탁발한 없음피부 뜨겁고 건조
  • 시원한 곳 이동 → 119 신고 → 목·겨드랑이·사타구니 냉각(얼음·냉수)

  • 지혈대는 최후 수단
저혈당 (Hypoglycemia)
식은땀손떨림어지러움의식 혼탁
